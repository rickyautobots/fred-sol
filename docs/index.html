‚ö° How It Works

Scan Markets

FRED scans Solana DEXs and prediction markets for trading opportunities using Jupiter and Birdeye APIs.

Estimate Probability

Each opportunity is analyzed by Claude, which returns a probability estimate and confidence level based on market context.

Calculate Position Size

Kelly criterion determines optimal bet size: f* = (bp - q) / b. We use half-Kelly with confidence adjustment.

Execute Trade

If edge exceeds 5%, FRED executes via Jupiter aggregator with built-in slippage protection.

Learn & Repeat

Outcomes are stored in persistent memory. Future decisions are adjusted based on historical performance.
